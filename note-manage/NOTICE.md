Mô hình cốt lõi: MVC (Model - View - Controller)
M Đảm nhận vai trò thiết kế dữ liệu và tương tác trực tiếp với Database.
V Chịu trách nhiệm về giao diện hiển thị cho người dùng cuối (Frontend nguyên bản với HTML/CSS/JS).
C Xử lý những logic nghiệp vụ (business logic) phức tạp, điều hướng dữ liệu từ Model trả về cho View.
Chuẩn giao tiếp: RESTful API
Mọi dữ liệu trao đổi qua lại giữa Frontend và Backend đều dưới định dạng JSON.
Framework nền tảng: Laravel
